# Global Signal Map

## 시스템 아키텍처 문서

---

# 1. 시스템 개요

Global Signal Map은 전쟁, 경제, 자연재해, 정치 등 세계 주요 이슈를 **지도 기반으로 시각화하는 웹 서비스**이다.

사용자는 세계 지도 위에 표시된 뉴스 핀을 통해 현재 발생하는 글로벌 이슈를 직관적으로 확인할 수 있으며, AI 분석을 통해 뉴스의 의미와 향후 영향을 쉽게 이해할 수 있다.

또한 사용자는 관심 있는 기사를 저장하여 **나만의 기사(My Articles)** 페이지에서 관리할 수 있다.

---

# 2. 시스템 목표

본 시스템은 다음 기능 제공을 목표로 한다.

* 지도 기반 세계 뉴스 시각화
* 카테고리별 이슈 탐색
* 뉴스 중요도 시각화
* 뉴스 검색 기능
* AI 기반 뉴스 분석
* 사용자 기사 저장 기능

---

# 3. 전체 시스템 아키텍처

```text
사용자 (Web Browser)
        │
        ▼
Frontend Web Application
(Next.js / React)
        │
        ▼
API Server
(Node.js / FastAPI)
        │
 ┌───────────────┬───────────────┬───────────────┐
 ▼               ▼               ▼
News API      AI Service      Database
(외부 뉴스)    (AI 분석)       (저장 데이터)
```

---

# 4. 시스템 구성 요소

## 4.1 Frontend (웹 클라이언트)

사용자 인터페이스와 지도 시각화를 담당한다.

### 주요 기능

* 세계 지도 렌더링
* 뉴스 핀 표시
* 뉴스 검색 기능
* 기사 저장 기능
* AI 분석 결과 표시

### 주요 기술

* Next.js
* React
* TailwindCSS
* Mapbox 또는 Leaflet

---

## 4.2 Backend API 서버

Frontend와 데이터 시스템 사이에서 데이터를 처리하는 서버이다.

### 주요 역할

* 뉴스 데이터 제공
* 뉴스 검색 처리
* 기사 저장 처리
* AI 분석 요청 처리
* 뉴스 중요도 계산

### 주요 API

* 뉴스 조회 API
* 뉴스 검색 API
* 기사 저장 API
* 기사 삭제 API

### 기술

* Node.js
  또는
* Python FastAPI

---

## 4.3 뉴스 데이터 수집 시스템

외부 뉴스 API에서 글로벌 뉴스를 수집한다.

### 데이터 출처 예시

* NewsAPI
* Reuters
* BBC
* Al Jazeera

### 수집 데이터

* 뉴스 제목
* 뉴스 요약
* 언론사
* 발행일
* 국가 / 지역
* 카테고리
* 좌표 정보
* 뉴스 중요도

---

## 4.4 AI 분석 서비스

뉴스 내용을 분석하여 사용자에게 쉽게 이해할 수 있는 정보를 제공한다.

### 제공 기능

1. 뉴스 요약
2. 상황 해석
3. 예상 동향 분석
4. 영향 분석

### 영향 분석 예시

전쟁

* 금 가격
* 석유 가격
* 주식 시장

경제

* 환율
* 금리
* 시장 영향

자연재해

* 공급망 영향
* 교통 영향

정치

* 외교 관계
* 무역 영향

---

## 4.5 데이터베이스

뉴스 데이터 및 사용자 데이터를 저장한다.

### 저장 데이터

뉴스 데이터

* 뉴스 ID
* 제목
* 요약
* 카테고리
* 위치 좌표
* 중요도

사용자 데이터

* 저장한 기사
* 저장 시간
* 기사 카테고리

### 데이터베이스 기술

* PostgreSQL
  또는
* MongoDB

# 5. 주요 기능 아키텍처

## 5.1 지도 기반 뉴스 표시

```text
뉴스 API
   │
   ▼
Backend 서버
   │
   ▼
뉴스 좌표 데이터 생성
   │
   ▼
Frontend 지도 렌더링
   │
   ▼
뉴스 핀 표시
```

뉴스 중요도에 따라 핀의 크기와 색상이 변경된다.

예시

* 높은 중요도 → 큰 핀
* 중간 중요도 → 중간 핀
* 낮은 중요도 → 작은 핀

---

# 5.2 뉴스 검색 기능

```text
사용자 검색 입력
       │
       ▼
Frontend 요청
       │
       ▼
Backend 검색 API
       │
       ▼
뉴스 데이터 반환
       │
       ▼
지도 + 뉴스 리스트 표시
```

사용자는 키워드를 통해 원하는 뉴스를 검색할 수 있다.

---

# 5.3 기사 저장 기능

```text
사용자 기사 저장 클릭
       │
       ▼
Frontend 요청
       │
       ▼
Backend API
       │
       ▼
Database 저장
       │
       ▼
My Articles 페이지 표시
```

사용자는 관심 있는 뉴스를 개인 페이지에 저장할 수 있다.

# 9. 결론

Global Signal Map은
**지도 기반 뉴스 시각화 + AI 분석 + 개인화 기능**을 결합한 글로벌 이슈 탐색 플랫폼이다.
