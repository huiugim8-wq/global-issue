# 세계 이슈 지도 플랫폼 요구사항 정의서

## 1. 프로젝트 개요

전쟁, 경제, 자연재해, 정치 등 세계 주요 이슈를 지도 기반으로 시각화하여 사용자가 세계 상황을 한눈에 파악할 수 있도록 하는 웹 서비스이다.
또한 사용자가 관심 있는 기사를 저장해 다시 볼 수 있는 **나만의 기사 기능**을 제공한다.

---

## 2. 서비스 구조

서비스는 다음 6개 페이지로 구성된다.

* 홈
* 전쟁
* 경제
* 자연재해
* 정치
* 나만의 기사

---

## 3. 홈 기능

홈은 세계 이슈를 종합적으로 보여주는 메인 대시보드 역할을 한다.

### 3.1 세계 이슈 지도

* 세계 지도 위에 주요 이슈를 핀 형태로 표시한다.
* 핀 색상을 통해 카테고리를 구분한다.

예시

* 전쟁: 빨강

* 경제: 초록

* 정치: 노랑

* 자연재해: 주황

* 사용자는 핀 클릭 시 뉴스 요약 정보를 확인할 수 있다.

### 3.2 뉴스 검색 기능

* 사용자는 키워드를 입력하여 원하는 뉴스를 검색할 수 있어야 한다.
* 검색 결과는 리스트와 지도 핀으로 함께 확인할 수 있어야 한다.

### 3.3 주요 뉴스 카드

* 현재 가장 영향력 있는 뉴스 5개를 카드 형태로 제공한다.
* 카드에는 제목, 지역, 카테고리, 언론사, 요약 정보가 포함되어야 한다.

---

## 4. 카테고리 페이지 기능

전쟁, 경제, 자연재해, 정치 페이지는 해당 주제의 뉴스만 집중적으로 보여준다.

### 4.1 지도 기반 뉴스 표시

* 해당 카테고리의 뉴스만 지도에 핀으로 표시한다.
* 뉴스 중요도에 따라 핀의 크기, 색상 강조, 시각 효과를 다르게 표현한다.
* 이를 통해 사용자는 중요한 뉴스를 직관적으로 구분할 수 있어야 한다.

예시

* 중요도 높은 뉴스: 큰 핀, 진한 색상
* 중요도 보통 뉴스: 중간 크기 핀
* 중요도 낮은 뉴스: 작은 핀, 옅은 색상

### 4.2 뉴스 상세 정보

* 핀 클릭 시 뉴스 상세 정보를 확인할 수 있어야 한다.

포함 정보

* 뉴스 제목
* 발생 지역
* 언론사
* 발행일
* 뉴스 요약

### 4.3 AI 분석 정보

뉴스 선택 시 다음 정보를 제공한다.

* AI 해석
* 예상 동향
* 미치는 영향

영향 예시

* 전쟁: 금, 석유, 주식
* 경제: 환율, 금리, 시장
* 자연재해: 공급망, 교통, 산업
* 정치: 외교, 무역, 시장 심리

### 4.4 기사 저장 기능

* 사용자는 카테고리 페이지에서 원하는 뉴스를 저장할 수 있어야 한다.
* 저장 버튼을 통해 관심 기사를 나만의 기사 페이지로 보낼 수 있어야 한다.

---

## 5. 나만의 기사 페이지 기능

나만의 기사 페이지는 사용자가 저장한 뉴스만 모아보는 개인화 페이지이다.

### 주요 기능

* 저장한 기사 목록 확인
* 기사 제목, 카테고리, 지역, 언론사, 요약 정보 표시
* 저장한 기사 상세 보기
* 저장 기사 삭제 기능
* 카테고리별로 저장 기사 분류 가능

### 목적

* 사용자가 관심 있는 뉴스만 따로 모아 다시 볼 수 있도록 한다.
* 단순 조회형 서비스가 아니라 개인 맞춤형 탐색 경험을 제공한다.

---

## 6. 데이터 요구사항

뉴스 데이터는 외부 뉴스 API를 통해 수집한다.

필요 데이터

* 뉴스 제목
* 언론사
* 발행일
* 뉴스 요약
* 국가 또는 지역
* 카테고리
* 지도 좌표
* 뉴스 중요도 정보

AI 분석 데이터

* 뉴스 해석
* 예상 동향
* 영향 분석

사용자 저장 데이터

* 저장한 기사 ID
* 저장 시간
* 기사 카테고리

---

## 7. MVP 기능

최소 구현 기능은 다음과 같다.

* 홈 대시보드
* 지도 핀 표시
* 핀 색상 카테고리 구분
* 뉴스 검색 기능
* 카테고리별 뉴스 페이지
* 중요도에 따른 핀 크기/강조 차등 표현
* 기사 저장 기능
* 나만의 기사 페이지
* AI 분석 정보 제공

---

이제 구조가 훨씬 서비스답다.

핵심은 세 가지야.

**첫째**, 홈은 전체 흐름을 보는 곳
**둘째**, 카테고리 페이지는 중요도를 시각적으로 보여주는 곳
**셋째**, 나만의 기사 페이지는 사용자가 관심 뉴스만 쌓아두는 곳

이 정도면 그냥 “뉴스 지도”가 아니라 **개인화된 세계 이슈 탐색 서비스**처럼 보여.

다음엔 이걸 바탕으로 **페이지별 화면 구성만 딱 보이게 와이어프레임식으로** 정리해줄게.
